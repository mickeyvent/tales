# Tales

calcula las semejanzas con el teorema de tales.

## Git clone 🚀

_$git clone https://github.com/mkyvent/tales.git._


### Pre-requisitos 📋

_Golang 1.19.2_

```
  Linux:
     apt install golang -y
  Termux:
     pkg install golang -y
```

### Instalación 🔧

_go run tales.go_
